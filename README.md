# 💱 Telegram-бот: Конвертер валют

Простой, быстрый и отзывчивый асинхронный Telegram-бот для конвертации валют в реальном времени. Написан на Python с использованием современных асинхронных библиотек.

## 🛠 Стек технологий
* **Python 3.10+**
* **aiogram 3.x** — асинхронный фреймворк для работы с Telegram Bot API
* **aiohttp** — асинхронные HTTP-запросы для получения актуальных курсов валют
* **asyncio** — управление асинхронным выполнением кода

## 🚀 Функционал
* Конвертация валют в один клик.
* Интуитивный интерфейс с использованием клавиатур.
* Быстрое переключение направлений обмена (прямой и обратный курс).
* Парсинг актуальных данных о курсах через внешнее API без задержек.
